# ShiftLink デザイン方針

最終更新: 2026-05

---

## デザインコンセプト

**「高級感 × 使いやすさ」**

メンズエステという業種の雰囲気（落ち着き・高級感・プロフェッショナル）を反映しつつ、
オーナー・スタッフが毎日使う業務ツールとして操作性を最優先にする。

---

## カラーパレット

### メインカラー
| 用途 | カラー | コード |
|------|--------|--------|
| プライマリ（ゴールド） | アクセント・ボタン・見出し | `#b8975a` |
| ダーク（ディープブラック） | ヘッダー・テキスト | `#1a1612` |
| バックグラウンド | ページ背景 | `#f9f7f4` |
| ホワイト | カード・シート背景 | `#ffffff` |

### サブカラー
| 用途 | カラー | コード |
|------|--------|--------|
| 成功・確定 | グリーン | `#4caf50` |
| 警告・注意 | オレンジ | `#ff9800` |
| エラー・削除 | レッド | `#f44336` |
| 情報・通知 | ブルー | `#2196f3` |
| ボーダー・区切り | ライトグレー | `#dddddd` |
| サブテキスト | グレー | `#888888` |

### カラー使用方針
- ゴールドはアクセントのみ。多用しない。
- 背景は `#f9f7f4`（生成り）を基調とし、清潔感と高級感を両立
- ダークモードは現状対象外

## タイポグラフィ

```css
font-family: 'Helvetica Neue', Arial, 'Hiragino Kaku Gothic ProN', sans-serif;
```

| 用途 | サイズ | ウェイト |
|------|--------|---------|
| ページタイトル | 20px | 600 |
| セクション見出し | 16px | 600 |
| 本文 | 14px | 400 |
| サブテキスト・注記 | 12px | 400 |
| 小ラベル | 11px | 400 |

- 行間: `line-height: 1.6〜1.8`
- 日本語は読みやすさ優先、英数字は Helvetica Neue

## レイアウト原則

### モバイルファースト
- 基本想定: スマートフォン縦持ち（375px〜）
- PC対応: 768px以上でmax-width 680px、1024px以上で780px
- 管理画面など情報量が多いタブは `pc-wide` クラスで幅を拡張

### 画面構造
```
[ヘッダー: 固定]
[コンテンツエリア: スクロール]
[ボトムナビ: 固定]
```

### ボトムシート
- 詳細入力・確認操作はボトムシート（`sheet-xxx`）で行う
- 背景は半透明オーバーレイ
- `border-radius: 16px 16px 0 0` で上部を丸く
- シート内はスクロール可能に

## UIコンポーネント方針

### ボタン
```css
/* プライマリボタン */
background: #b8975a;
color: white;
border-radius: 8px;
padding: 12px 20px;
font-size: 15px;

/* セカンダリボタン */
background: transparent;
border: 1px solid #b8975a;
color: #b8975a;

/* 危険操作ボタン */
background: #f44336;
color: white;
```

- タップ領域は最低 44×44px 確保
- 危険な操作（削除等）は赤ボタン + confirmダイアログ必須

### カード・リスト
- `border-radius: 10px`、`box-shadow: 0 2px 8px rgba(0,0,0,0.08)`
- カード間のマージン: 10〜12px
- タップ時: `background: #f5f5f5` でフィードバック

### トースト通知
- 画面下部中央に表示
- 成功: グリーン背景
- エラー: レッド背景
- 表示時間: 3秒（自動消去）

### フォーム
- ラベルはフォームの上に配置（フローティングラベルは使わない）
- `border: 1px solid #ddd`、フォーカス時 `border-color: #b8975a`
- `border-radius: 6px`

---

## アイコン方針

- **Material Icons**（Google Fonts）をCDN経由で使用
- サイズ: 24px 基本、ナビゲーションは 22px
- カラー: テキストと同じか `#888888`（サブアイコン）

---

## アニメーション

- ボトムシートの開閉: `transform: translateY` + `transition: 0.3s ease`
- タブ切替: フェードなし（即時切替、速さ優先）
- トースト表示: `fadeIn` アニメーション 0.2s
- 重いアニメーションは禁止（業務ツールのため動作速度優先）

---

## 禁止事項

- ❌ DOM要素の削除による非表示（`display:none` も基本NG）→ `height:0; overflow:hidden` を使う
- ❌ 装飾目的の複雑なCSS（グラデーション多用、複雑なシャドウ等）
- ❌ フォントの多用（ファミリーは1種のみ）
- ❌ スクロールを妨げる固定要素の多用

---
