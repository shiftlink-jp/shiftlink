# ShiftLink LP 画像生成プロンプト集（ChatGPT用・2026-06-25）

ChatGPTにそのまま1つずつ貼って画像を作るための、独立した生成プロンプト10枚分。
各プロンプトは単体で完結（スタイル指定を毎回含む）。トーンを10枚で揃えるため、共通スタイルは各プロンプトに同じ文を入れてあります。

## 使い方
1. 下の「①」〜「⑩」のコードブロックを1つずつコピーしてChatGPTに貼る。
2. 生成された画像を、各見出しの `保存名` でダウンロード。
3. 10枚揃ったら開発部に渡す → LP差し替え。

## 重要な注意（日本語テキストについて）
- 各プロンプトに「画像内に入れる日本語テキスト」を明記しています。**誤字なく・読みやすいゴシック体で**と指示済み。
- もしChatGPTが日本語の文字を崩す場合は、**プロンプト末尾の「※文字は入れず、テキスト用の余白だけ作って」という一文に切り替えて画像だけ作り、文字はCanva等で後載せ**してください（崩れ防止）。各プロンプトにその代替指示も併記しています。
