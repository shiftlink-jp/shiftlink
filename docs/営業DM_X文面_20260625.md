# ShiftLink 営業DM・X発信 文面集（2026-06-25）

教材8本＋営業状況（LINE56件→契約0）を踏まえた、営業DM・X発信の文面。
ABテスト前提で複数パターン用意。社長がそのまま使える粒度。

## 大前提（なぜ前回0件だったか／今回の方針）
- 前回失敗の真因＝**いきなり本命（月額ツール）を売ろうとした**（2ステップ不在）。固定費アレルギーで即スルー。
- 今回の方針：
  1. **売り込まない。2ステップにする** … DMのゴールは「契約」でなく「無料で試す/役立つ情報を受け取る」だけ。
  2. **本音を突く** … 「ツール使いませんか」でなく「委託金/数字の不安」に共感から入る。
  3. **手間ゼロを約束** … データ移行を全部代行（乗り換えの壁を壊す）。
  4. **1メッセージ1オファー** … CTAは1つだけ。返信のハードルを最小に。
  5. **制限** … 先着10店舗・初期設定無料＋移行代行。
- ※法令：誇大・虚偽表現は禁止。特定電子メール法等に配慮し、しつこい連投はしない。

## 1. 営業DM（コールド）— ABテスト用4パターン

### パターンA（本音・共感型）★まず試す
```
突然のご連絡失礼します。メンズエステ店さん向けに、シフト・予約・売上・委託金をスマホひとつで管理できるツールを作っている者です。

売り込みではなく、ひとつだけ。
「売上が落ちた時、"なぜ落ちたか"を数字でパッと説明できる状態」って、今ありますか？

もしそこに少しでも引っかかったら、14日間無料（カード登録なし）で店の数字が見える状態をお試しいただけます。合わなければ放っておくだけで課金もありません。

ご興味あれば「見てみたい」とだけ返信ください。30秒のデモをお送りします。
```

### パターンB（委託金・キャスト定着型）
```
はじめまして。メンエス店さん専門の管理ツールを作っている者です。

いきなりですが、委託金（バック）の計算でキャストさんと気まずくなったこと、ありませんか？
1円のズレが信頼に効いてくる世界なので、ここを自動で正確に出せるツールを作りました。

今、先着10店舗さま限定で「初期設定＋他社ツールからのデータ移行も全部こちらで無料代行」しています。乗り換えの面倒は、ゼロです。

14日間無料・カード登録なしで試せます。気になれば「詳しく」とだけ返信ください。
```

### パターンC（乗り換えの壁を壊す型）※既に他ツール利用の店向け
```
突然失礼します。メンエス店さん向けの管理ツールを作っている者です。

「今のツールでも別に困ってないし、乗り換えるの面倒だし…」
よく言われます。なのでお伝えしたいのは1点だけ。

データの移行、こちらで全部やります。あなたの手間はほぼゼロです。
その上で14日間無料（カード登録なし）。今のツールと並べて比べてから決めていただけます。

委託金の自動計算と、売上の見える化はたぶん気に入っていただけます。
「比べてみたい」と返信いただければ、デモをお送りします。
```

### パターンD（情報提供・リードマグネット型）※売らずに価値だけ渡す
```
はじめまして。メンエス店さんの売上・キャスト管理を研究している者です。

「委託金トラブルでキャストを辞めさせないためのチェックリスト」を作ったので、よければ無料でお渡しします（営業資料ではなく、純粋に現場で使えるものです）。

受け取りたい方は「ほしい」とだけ返信ください。
※もしツール化したものにも興味が出たら、その時に14日間無料でお試しいただけます。
```

### フォロー文面（前回「検討します」と返した2件向け）
```
先日は「検討します」とお返事ありがとうございました。その後いかがでしょうか。

ひとつだけ良いお知らせで、今なら先着10店舗さま限定で「初期設定＋データ移行を全部無料代行」しています。乗り換えの手間がネックでしたら、そこは完全にこちらで巻き取ります。

14日間無料・カード登録なしなので、まず数字を見るだけでもいかがですか？ ご都合の良いタイミングでデモをお送りします。
```

### DM運用メモ
- 4パターンを各10〜15件ずつ送り、**返信率が高いものを残す**（=ABテスト。木下・三崎の「数字で選ぶ」）。
- 1通目で売り切らない。返信が来たら「30秒デモ→無料トライアル」へ自然に誘導（2ステップ）。
- CTAは毎回1つ・1単語で返せる形（「見てみたい」「詳しく」「ほしい」）＝申込の楽さ最優先。
- 連投・しつこさは厳禁（ブランド毀損／西野「とどめを刺さない」）。

## 2. X（旧Twitter）発信 — 認知より"深さ"を作る運用

### 方針（西野・木下）
- バズ狙いでなく、**メンエス経営者が「この人わかってる」と感じる発信**で信頼（人気＝深さ）を積む。
- 広告/DMで知った人がXを見て安心→LPへ、の**メディアミックス5回接触**を作る。
- 「数字で語る」発信（木下）。役立つ→たまにShiftLinkの話。

### コンテンツの柱（4本柱を回す）
1. **委託金・キャスト定着**の知見（最重要・重心）
2. **売上の数字の見方**（広告ROI・客単価・指名率）
3. **メンエス経営あるある／本音への共感**
4. **ShiftLinkの裏側・先行募集の告知**（売り込みは控えめに）

### 投稿サンプル（そのまま使える・適宜調整）

**① 共感×本音**
```
メンエス経営でいちばん消耗するの、施術でもシフトでもなくて
「お金の計算でキャストと気まずくならないように気を使うこと」だと思う。

委託金が1円ズレるだけで信頼が削れる。
ここを自動で正確に出せるだけで、辞める人が減る。
```

**② 数字の見方（役立つ）**
```
「今月売上落ちた…」を感覚で語る人が多いけど、原因は3つに分解できる。
①来店数 ②指名率 ③客単価
このどれが落ちたかを見れば、打つ手が変わる。
"なんとなく不調"がいちばん危ない。
```

**③ 乗り換えの壁（移行代行）**
```
「今のツール、別に不満ないけど乗り換えるの面倒」
わかります。その"面倒"が現状維持の正体。

なのでうちは、データ移行を全部こっちでやることにした。
店側の手間ゼロ。比べてから決めてもらえばいい。
```

**④ あるある共感**
```
メンエス経営、相談できる相手が驚くほどいない。
同業に手の内は明かせないし、税理士はこの業界の中身まで知らない。
数字もキャストも、結局ひとりで抱える。
この"孤独"を軽くしたくてツールを作ってる。
```

**⑤ 先行募集の告知（制限）**
```
メンエス専門の管理ツール、いま最初の10店舗さまを募集中。
・初期設定 無料サポート
・他社ツールからのデータ移行も全部無料代行
・14日間無料（カード登録なし）

"最初の一歩"はこちらで伴走します。気になる方はDMください。
```

**⑥ ビフォーアフター（具体）**
```
紙・LINE・エクセルに散らばってた管理を、スマホ1台に集約したら何が変わるか。
・委託金が自動で出る → 揉めない
・売上がグラフで見える → 効いてない広告に気づく
・シフトのLINE往復が消える
"作業"が減って"経営"に時間が戻る。
```

### X運用メモ
- 1日1〜2投稿、柱①〜④を回す。⑤告知は週1程度（売り込みすぎない）。
- プロフィール／固定ポストにLPリンク＋「先着10店舗募集中」を置く（広告で来た人の受け皿＝5回接触）。
- 反応が良かった投稿の型を残す（ABテスト思考）。
- 数字や事例は事実のみ（契約0の現状で「導入◯店」等の虚偽は書かない）。
